Get a delayed response from the server for latency testing.
Will return a response after :delay milliseconds. Latency in the network connection is not accounted for, so you will receive a response in :delay + latency milliseconds

Request

GET /:delay

Parameters

delay required
integer

Response

Status: 200 OK 
{
  "status": "success",
  "delay": 1000
}
